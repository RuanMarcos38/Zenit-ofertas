ZÊNITE OFERTAS - ARQUIVOS PRONTOS PARA HOSTINGER

1) Entre no painel da Hostinger.
2) Acesse: Sites > Gerenciar > Gerenciador de Arquivos.
3) Abra a pasta public_html do seu domínio.
4) Envie estes arquivos:
   - index.html
   - .htaccess
5) Confirme a substituição dos arquivos antigos.
6) Acesse seu domínio em uma guia anônima para conferir.

IMPORTANTE:
- O arquivo index.html já tem CSS, JavaScript e a logo incorporados.
- Não precisa subir pasta de imagens.
- Todos os botões já estão direcionando para o Grupo VIP do WhatsApp.
- O site é responsivo para celular, tablet e computador.

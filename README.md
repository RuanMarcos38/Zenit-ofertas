# Zênite Ofertas

Landing page profissional da Zênite Ofertas, pronta para hospedagem simples em Hostinger, cPanel, Apache ou qualquer servidor estático.

## Arquivos

- `index.html` — site completo da Zênite Ofertas.
- `.htaccess` — configuração para Apache/Hostinger com HTTPS, segurança básica e cache.
- `README-HOSTINGER.txt` — passo a passo rápido para publicação.

## Como publicar na Hostinger

1. Acesse o painel da Hostinger.
2. Abra o Gerenciador de Arquivos.
3. Entre na pasta `public_html` do domínio.
4. Envie os arquivos `index.html` e `.htaccess`.
5. Substitua arquivos antigos, se existirem.
6. Limpe o cache do navegador e acesse o domínio.

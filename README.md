# Zênite Ofertas

Página de captura profissional da Zênite Ofertas.

## Estrutura

- `index.html` — página principal em sessão única.
- `assets/zenite-logo.webp` — logo oficial usada no Hero.
- `.htaccess` — configuração para Hostinger/Apache.
- `README-HOSTINGER.txt` — instruções de publicação.

## Publicação

Envie todos os arquivos para `public_html` na Hostinger, mantendo a pasta `assets`.
